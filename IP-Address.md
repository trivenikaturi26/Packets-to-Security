## What is an IP Address?

Before we get into the technical definition, here's a fun way to think about it:

**An IP Address is like an Aadhaar number for a device.**

Just as every citizen has a unique Aadhaar number for identification, every device connected to a network is assigned an IP address so it can be identified and communicate with other devices.

Of course, the comparison isn't perfect, but it makes the basic idea easier to remember. 😄

When we want to call someone, we use their phone number. Similarly, when devices communicate over a network, they use an IP address.

An IP Address (Internet Protocol Address) is a unique identifier assigned to a device connected to a network. It helps devices find and communicate with each other.

Some examples of IP addresses are:

```text
192.168.1.1
8.8.8.8
```

Without IP addresses, sending data across networks would be impossible because devices would not know where to send or receive information.

---

## Why Are IP Addresses Important?

Every activity we perform on the Internet depends on IP addresses.

For example:

- Opening a website
- Watching YouTube videos
- Sending emails
- Playing online games

In all these cases, data travels between devices using IP addresses.

---

## Types of IP Addresses

### Public IP Address

A Public IP Address is assigned by an Internet Service Provider (ISP).

This address is visible on the Internet and allows communication with external networks.

Example:

```text
49.xxx.xxx.xxx
```

### Private IP Address

A Private IP Address is used inside a local network such as a home, college, or office network.

Examples:

```text
192.168.1.10
10.0.0.5
```

Private IP addresses cannot be accessed directly from the Internet.

---

## IPv4

IPv4 (Internet Protocol Version 4) is the most widely used version of IP.

Characteristics:

- Uses 32 bits
- Written in dotted decimal format
- Supports around 4.3 billion addresses

Example:

```text
192.168.1.1
```

---

## IPv6

As the number of Internet-connected devices increased, IPv4 addresses started becoming limited.

IPv6 was introduced to solve this problem.

Characteristics:

- Uses 128 bits
- Provides a huge number of unique addresses
- Designed for the future growth of the Internet

Example:

```text
2001:db8::1
```

---

## Public vs Private IP Address

| Public IP |

*Visible on the Internet 
*Assigned by ISP 
* Globally unique


|Private IP |

*Used inside local networks 
*Assigned by router 
*Can be reused in different networks 

---

## Real-World Example

Suppose I connect my phone to my home Wi-Fi.

My phone may receive a private IP address like:

```text
192.168.1.5
```

When I open Google, my router uses its public IP address to communicate with Google's servers and then sends the response back to my phone.

This entire process happens within seconds.

## What I Learned

- Every device on a network needs an IP address.
- IP addresses help identify and locate devices.
- IPv4 uses 32 bits, while IPv6 uses 128 bits.
- Public IP addresses are used on the Internet.
- Private IP addresses are used within local networks.
